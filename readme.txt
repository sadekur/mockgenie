== Description ==

MockGenie allows you to generate and edit images directly within your WordPress site. 
Simply go to your media library and use AI-powered tools to create images without leaving WordPress.

== Installation ==

1. Upload the `mockgenie` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your API key in the plugin settings.
